LiChess Games Analysis

Cleaning up Data

Lichess will only allow pgn files to be downloaded. In order to use the data, I first exported the pgn data to text.



Then, using Power Query, I cleaned up the data so that each tasty morsel of data had its own column.



For our next step, we load the csv file into Microsoft SQL Server and start running some interesting queries.



Now that we know how many games of each opening are played, we can make simple charts or graphs that tell a little story.
